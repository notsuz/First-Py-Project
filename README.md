###  Description
A professional web application demonstrating advanced backend architecture and responsive frontend design. This project focuses on scalable data management and clean user interface principles.

###  Features
* ** Secure Auth:** Full user registration and login functionality.
* ** Responsive UI:** Mobile-first design using Bootstrap 5 components.
* ** Dual-DB Support:** Configured for local development (SQLite) and production (Postgres).
* ** Optimized Queries:** Efficient data handling using Django's ORM.
